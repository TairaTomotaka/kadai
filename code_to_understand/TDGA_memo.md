* [学習の正則化](https://zero2one.jp/learningblog/yobinori-collab-regularization/)
    * 学習の正則化とは、過学習を防ぐ手法の一つである. 
    * $E'(w) = E(w) + \lambda R$  $(\lambda > 0)$ を最小化. 
    * つまり, $R$ という制約のもとで $E(w) $ を最小化. 
    * L2 正則化: $R = \Sigma_j w_j^2 $
    * L1 正則化: $R = \Sigma |w|$
